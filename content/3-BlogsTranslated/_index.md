### [Blog 1 - Experiencing Amazon Nova Act: AI-Powered UI Automation Without XPath Worries](3.1-Blog1/)

This blog shares hands-on experience with Amazon Nova Act, a new AI Automation service by AWS that enables building Agents to interact with browsers using natural language in English. It details how the service leverages Reinforcement Learning, explains how to write hybrid code (combining Python SDK with AI commands), and covers secure deployment on cloud Fleet infrastructure.


### [Blog 2 - Building Secure B2C Applications with Amazon Cognito and Amazon Verified Permissions](3.2-Blog2/)

This blog introduces a solution for building secure B2C applications by combining Amazon Cognito and Amazon Verified Permissions to completely decouple the authentication and authorization processes. You will learn how to overcome the drawbacks of hardcoding authorization logic directly into your application by defining flexible policies using the Cedar language. The article also provides a detailed analysis of the architectural flow, supported authorization models, testing methods, and considerations for optimizing costs as the system scales.

### [Blog 3 - Creating 3D Models by Using AI to Convert 2D Images into 3D Assets (AWS)](3.3-Blog3/)

This blog guides you through the comprehensive process of creating 3D assets from 2D images by deploying open-source AI models such as TripoSG and MV-Adapter on AWS infrastructure. You will explore a solution architecture that utilizes Amazon S3 as a centralized storage repository and Amazon EC2 GPU instances to optimize performance for 3D mesh generation and texturing tasks. The post also provides practical steps for handling mesh errors, optimizing models before importing them into game engines, and tips for saving on cloud computing costs.
