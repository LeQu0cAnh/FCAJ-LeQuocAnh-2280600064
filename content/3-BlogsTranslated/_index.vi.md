### [Blog 1 - Trải nghiệm Amazon Nova Act: Tự động hóa UI bằng AI, không lo fix XPath](3.1-Blog1/)

Blog này chia sẻ trải nghiệm thực tế về Amazon Nova Act, một dịch vụ AI Automation mới của AWS giúp xây dựng các Agent tự động tương tác trình duyệt thông qua câu lệnh ngôn ngữ tự nhiên tiếng Anh. Bài viết đi sâu phân tích cơ chế hoạt động bằng Học sâu tăng cường (Reinforcement Learning), hướng dẫn các bước lập trình hybrid (kết hợp Python SDK và AI lệnh) và phương pháp deploy mở rộng hạ tầng Fleet an toàn trên Cloud.

### [Blog 2 - Xây dựng ứng dụng B2C an toàn với Amazon Cognito và Amazon Verified Permissions](3.2-Blog2/)

Blog này giới thiệu giải pháp xây dựng ứng dụng B2C an toàn bằng cách kết hợp Amazon Cognito và Amazon Verified Permissions nhằm tách biệt hoàn toàn quá trình xác thực (authentication) và phân quyền (authorization). Bạn sẽ tìm hiểu cách khắc phục nhược điểm của việc viết logic phân quyền trực tiếp trong code bằng cách định nghĩa các policy linh hoạt thông qua ngôn ngữ Cedar. Bài viết cũng phân tích chi tiết luồng kiến trúc, các mô hình phân quyền được hỗ trợ, cách kiểm thử và những lưu ý để tối ưu chi phí khi hệ thống mở rộng.

### [Blog 3 - Tạo 3D Asset từ hình ảnh 2D bằng AI trên AWS](3.3-Blog3/)

Blog này hướng dẫn quy trình tạo 3D Asset hoàn chỉnh từ hình ảnh 2D bằng cách triển khai các mô hình AI mã nguồn mở như TripoSG và MV-Adapter trên hạ tầng AWS. Bạn sẽ tìm hiểu kiến trúc giải pháp sử dụng Amazon S3 làm kho lưu trữ trung tâm và Amazon EC2 GPU để tối ưu hóa hiệu năng cho các tác vụ sinh mesh 3D và phủ texture. Bài viết cũng cung cấp các bước thiết thực để xử lý lỗi mesh, tối ưu hóa mô hình trước khi đưa vào game engine, và các mẹo tiết kiệm chi phí tính toán đám mây.
